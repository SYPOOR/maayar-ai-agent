## Export

For presentation decks, ASCII diagrams can be rendered to PNG/SVG via terminal capture or diagram tooling. High-fidelity exports are stored in [assets/diagrams/](../../assets/diagrams/README.md).

## Conventions

| Symbol | Meaning |
|--------|---------|
| `───▶` | Data/control flow |
| `┌─┐` | System boundary or layer |
| `│` | Vertical connection |
| `▼` | Downward flow |
