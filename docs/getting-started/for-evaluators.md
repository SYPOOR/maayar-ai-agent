## Evaluation Signals

| Signal | Where to Find |
|--------|---------------|
| Architectural maturity | Layered design, ADRs, component boundaries |
| Engineering discipline | Output contracts, workflows, operational docs |
| Product clarity | Use cases, value proposition, roadmap |
| IP posture | Documentation-first strategy, clear NOTICE |
