# FAQ

---

## What is Maayar?

An agricultural risk intelligence platform that produces structured, multi-dimensional risk assessments for farms and portfolios.

## Is this open source?

This repository contains public architecture documentation under the MIT License. Production implementations are private. See [NOTICE.md](../NOTICE.md).

## Why no source code?

A documented architectural publishing strategy that demonstrates engineering maturity while protecting proprietary logic. See [ADR-0004](../architecture/decisions/0004-documentation-first-repository.md).

## What is the Composite Risk Index?

The platform's primary output — a normalized risk indicator with dimensional decomposition and confidence metadata. See [Decision Support Layer](../architecture/decision-support-layer.md).

## What external data is integrated?

NASA POWER (climate) and CHIRPS (precipitation). See [integrations.md](../technology/integrations.md).

## Where is it deployed?

Microsoft Azure. See [deployment-topology.md](../architecture/deployment-topology.md).

## Can I contribute code?

No. Documentation contributions only. See [CONTRIBUTING.md](../CONTRIBUTING.md).
