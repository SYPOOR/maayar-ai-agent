# Glossary

Enterprise terminology for the Maayar platform.

---

| Term | Definition |
|------|------------|
| **Assessment Core** | Central lifecycle manager for risk assessment sessions |
| **Capability Domain** | Independent analysis module within the Intelligence Layer |
| **Classification Services** | Risk normalization, weighting, and tier assignment |
| **Composite Risk Index** | Normalized composite risk indicator for cross-farm comparison |
| **Data Acquisition Layer** | Data ingestion, enrichment, and caching |
| **Decision Support Layer** | Final assessment output with decomposition and provenance |
| **Domain Signal** | Standardized output object from a capability module |
| **Intelligence Layer** | Orchestration and parallel capability execution |
| **Risk Intelligence Pipeline** | End-to-end processing from acquisition to decision support |
| **Risk Tier** | Categorical classification: Low, Moderate, Elevated, Critical |
| **Provenance** | Data source and processing attribution metadata |
